PlateVote
Project Overview:
PlateVote is a mobile application to solve "where to eat" dilemma. Users create a session, invite friends via a code 
Core features include: 
- Real time lobby
- Live vote
- Smart results with tie break

Tech Stack: 
Frontend - React Native Expo with Typescript
Backend - Supabase (postgreSQL + Realtime)
Navigation - Expo router

What to Do to Run: 
- Install Node.js
- Install Expo Go app on your IOS or Android device for testing on hardware

1) Installation
First, if you haven't, clone repo and install dependencies through following commands:

git clone [url_here]
cd platevote
npm install

2) Run Expo dev server using following command:

npx expo start

3) View app:
- Scan the QR code displayed on the terminal using the Expo Go App (Android) or Camera (iOS)
- Emulator (press a for android or i for iOS)
